ğŸ“‚ Dataset Details
The dataset includes the following key columns:

Match ID: Unique identifier for each match.
Player Name: Name of the batsman.
Runs Scored: Total runs scored by the player in an innings.
Balls Faced: Number of balls faced by the batsman.
Strike Rate: Player's scoring efficiency (calculated as runs/balls * 100).
Boundaries: Total 4s and 6s hit by the batsman.
Match Date: The date on which the match was played.
Opponent Team: Team the player was batting against.
